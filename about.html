Creation of a QR Code Poster :


I designed an attractive and professional visual using advanced features in Photoshop. I carefully selected suitable colors, fonts, and arranged elements to capture the attention of event participants. The key element of our poster was the QR code, which I integrated to allow participants to easily and quickly scan it using their mobile phones to access the form.




Extract:


The platform features a functionality that allows downloading each response in the form of an Excel file. We leveraged this feature by downloading all survey responses into separate Excel files. To streamline the download process and save time, I developed a Python code using Selenium to extract data from the web application. This code provides a user-friendly interface prompting the user to specify the destination folder path for downloaded files, as well as the email address and password of the authorized user for file downloads. Additionally, it allows the user to select the dates of the files to be downloaded (multiple dates can be chosen) and specify the name of the relevant survey. Once all information is provided, the code handles downloading the files one by one to the specified folder. Through this automation, I simplified the file download process, providing a practical and efficient solution for collecting necessary data from the web application. These files were then used as data sources. I developed specific Python code to ensure the collection and consolidation of all data into a single Excel file. Using the os library, I traversed through all files to obtain a list containing the names of source files. This list allowed us to iterate through them individually using the openpyxl library to extract the necessary data.


Transform:


In this section, I grouped responses by question since each file contains a single response for each question. Subsequently, we calculated the occurrence of each response to transform them into percentages. For evaluation questions, I classified responses based on a predefined scale: 0 corresponds to "very poor," 1 to "poor," 2 to "average," 3 to "good," 4 to "very good," and 5 to "excellent." This classification provided a better understanding of the provided evaluations and allowed for a more precise analysis. By transforming responses into percentages and using a rating scale, we were able to represent the opinions expressed by event participants clearly and meaningfully.


Load:


Finally, I proceeded to load all this data into a consolidated Excel table. This step allowed me to gather all collected information into a single file, facilitating their management and subsequent analysis. Using tools such as the pandas library, I merged data from different sources into a coherent whole.
